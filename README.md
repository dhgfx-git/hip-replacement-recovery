# Hip Replacement Recovery Tracker

A personal recovery tracking tool built and used during hip replacement surgery (April 2026). Developed as a practical day-to-day aid to monitor pain levels, medication schedules and recovery progress.

## What's included

- `recovery-tracker.html` — standalone single file app with pain journal, medication log, daily notes and recovery guide. Runs entirely in the browser, no installation needed
- `recovery-journey.csv` — exported data log covering day of surgery through to current recovery

## Features

- Daily pain scoring (0–10)
- Medication tracking with dosage and timing
- Tagged journal entries
- Day by day recovery milestone guide

## Context

Built during the early days of recovery as a practical tool to track progress and communicate clearly with physiotherapists and medical staff. Data was used to generate visual recovery charts and identify trends throughout rehabilitation.
